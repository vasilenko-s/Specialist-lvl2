<ul>
<li>Работа с cookie</li>
<li>Сессии</li>
<li>Запись/чтение из файла</li>

<li>Прием-обработка параметров,данных из форм</li>
<li>Работа с многомерным массивом. Отрисовка результатов запроса, таблиц "циклом-в-цикле"</li>
